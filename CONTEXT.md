# Контекст проекта: File Share

## 📌 Описание
Веб-сервис для обмена файлами по ссылке. Загружаешь файл → получаешь ссылку → делишься с друзьями.

## 🏗 Архитектура
- **Backend:** Python 3.12, FastAPI, uvicorn
- **Frontend:** HTML + Bootstrap 5
- **Storage:** Локальная папка `uploads/` (файлы по UUID)
- **Без БД:** Метаданные не хранятся

## ✅ Реализовано (v1.0.1)
- [x] Загрузка файлов (POST /upload)
- [x] Скачивание по ссылке (GET /files/{id})
- [x] Веб-интерфейс (GET /)
- [x] Endpoint версии (GET /version)
- [x] Docker + docker-compose
- [x] GitHub репозиторий
